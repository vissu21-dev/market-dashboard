## How to Use Trade Command

### **Step 1: Open Trade Command Tab**
```
Click "🎯 Trade Command" in main navigation
```

### **Step 2: Check Market Overview**
```
See VIX level:
├─ < 18 ✓ Normal (green)
└─ > 18 ⚠️ High (red) - reduce position size
```

### **Step 3: Check ORB Signal**
```
See if market is:
├─ 🟢 BREAKOUT (above ORB High) → CE bias
├─ 🔴 BREAKDOWN (below ORB Low) → PE bias
└─ 🟡 INSIDE RANGE → Wait for breakout
```

### **Step 4: Read Technical Analysis**
```
Expander shows:
├─ Bull score vs Bear score
├─ 🟢 Bullish confirmations (RSI, trend, etc.)
├─ 🔴 Bearish confirmations
└─ 🟡 Neutral signals
```

### **Step 5: Select Index & Budget**
```
Radio button: Nifty 50 / Bank Nifty / Both
Input: How much ₹ per lot can you risk
```

### **Step 6: Review Recommended Strikes**
```
For RECOMMENDED direction (CE or PE):
├─ Look for ★ RECOMMENDED badge (OTM 1 level)
├─ Check if you can afford it
├─ Read the profit/loss per level
└─ Review risk/reward ratio
```

### **Step 7: Enter Trade**
```
Using the recommended strike:
├─ Entry: Current market price for that strike
├─ SL: Premium SL level shown (-30%)
├─ T1: +60% premium level
├─ T2: +120% premium level
├─ Exit time: 1:30 PM (before events)
└─ Max lots: As calculated by system
```

### **Step 8: Monitor Index Levels**
```
While in trade, watch:
├─ Index SL level (exit if crossed)
├─ Index T1 level (book 50%)
├─ Index T2 level (trail rest)
└─ Premium SL (-30%)
```

### **Step 9: Check Live Options Chain**
```
Below Trade Command:
├─ Real-time prices for all strikes
├─ Compare different levels
├─ Verify premiums before entry
└─ Monitor IV changes
```
